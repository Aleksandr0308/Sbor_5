Сборный проект-4
Вам поручено разработать демонстрационную версию поиска изображений по запросу.

Для демонстрационной версии нужно обучить модель, которая получает векторное представление изображения и векторное представление текста, а на выходе выдаёт число от 0 до 1 — показывает, насколько текст и изображение подходят друг другу.
